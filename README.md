### Functionality
- Create tasks
- Edit tasks
- Log in
- Log out
- Redirect to login page when not authenticated

### How to use
Assuming you have Tomcat set up in /Library/Tomcat, just run
```
gradle deploy
```  
Then, start Tomcat and point your browser to `http://localhost:8080/tasks`
