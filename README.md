# onix-challenge

### Functionality
- Create tasks
- Edit tasks
- Log in with Google account
- Log out
- Redirect to login page when not authenticated

### How to use
`gradle appengineRun`  
The main page is located at /tasks, so for a local run it would be `http://localhost:8888/tasks`
